# 📈 Adani Stock Data Analysis Dashboard

An interactive Power BI dashboard built to analyze **Adani Enterprises** stock market data. The project demonstrates data cleaning, transformation, DAX calculations, and interactive data visualization to generate meaningful business insights.

---

## 📌 Project Overview

This dashboard provides a comprehensive analysis of Adani Enterprises stock data using Power BI. The raw CSV dataset was first cleaned and formatted in Microsoft Excel before being imported into Power BI for data transformation and visualization.

---

## 🚀 Dashboard Features

- 📊 KPI Cards (Open, Close, 52 Week High, 52 Week Low & Volume)
- 📈 Daily Stock Price Trend
- 📅 Monthly Volume Analysis
- 📉 Quarterly High vs Low Comparison
- 🎯 Interactive Slicers (Day, Month, Quarter, Year)
- ⚡ Dynamic and Interactive Visualizations
